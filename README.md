<h1>概要</h1>

Reactの勉強を兼ねて、自己紹介ページを作成しました。<br>
是非、最後までサイトを楽しんでいただけたらと思います。<br>
<br>
離脱率を減少させるために、アニメーションを多く取り入れ飽きない工夫をしました。<br>
ページが遷移する際の滑らかさを意識し、シームレスなウェブ体験を演出しております。
